Bitcache for Drupal
===================
This is a beta version of the Bitcache module currently being developed for
Drupal 6.x under the auspices of the project at:

  <http://drupal.org/project/bitcache>

For more information about Bitcache, see the project's home page and mailing
list:

  <http://bitcache.org/>
  <http://groups.google.com/group/bitcache>


BUG REPORTS
-----------
Post bug reports and feature requests to the issue tracking system at:

  <http://drupal.org/node/add/project_issue/bitcache>
